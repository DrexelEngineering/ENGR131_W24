In this lab, you will install software to run Jupyter Notebooks locally so that you can complete your assignments on your own computer.

Please choose the appropriate subsection based on whether you use a computer with Windows, Mac, or Linux.

Windows#

  1. Open this link to download Anaconda in a browser tab.

  2. Follow the instructions in the documentation below.

from IPython.display import IFrame

In the window that pops up, look for the word Processor.

  • If the words following Processor include ‚ÄúM1‚Äù, ‚ÄúM2‚Äù, or ‚ÄúM3‚Äù, follow the instructions under #### M1-3 below.

C. Create an Environment for ENGR 131#

  1. Download this file to a location where you can find it. Do not change the file name, which should be engr131_24.yml.

  2. In Anaconda Navigator, click Environments on the left navigation panel.

  3. Click Import at the bottom of the window.

  4. Choose the file that you downloaded in Step 0. Leave the new environment name as engr131_24.

  5. Click Import.

  6. Make sure that engr131_24 is now in the list under ‚Äúbase(root)‚Äù and is highlighted in green. Click the name ‚Äúengr131_24‚Äù to select it if it is not green.

D. Launch JupyterLab#

  1. In Anaconda Navigator, click the Home button in the left navigation panel.

  2. Find JupyterLab (not Jupyter notebooks) in the applications panel.

  3. Click ‚ÄúLaunch‚Äù under JupyterLab.

E. Create the engr131 Kernel#

Windows#

  1. Go to Anaconda Navigator Home

  2. Look for the CMD.exe PROMPT application, and click Install beneath it.

  3. Launch CMD.exe PROMPT application by clicking Launch beneath it.

  4. In the command prompt window that opens, enter exactly the following at the prompts.

conda activate engr131_24

conda install ipykernel

ipython kernel install --user --name=engr131

conda deactivate

  1. Then, close the command prompt window. Proceed to Step F. Test JupyterLab with engr131 Kernel.

Mac or Linux#

  1. In your default browser, JupyterLab should be open.

  2. Click on ‚ÄúTerminal‚Äù to open a terminal.

  3. In the terminal window, enter the following at the prompts.

conda activate engr131_24

conda install ipykernel

ipython kernel install --user --name=engr131

conda deactivate

  1. Then, close the terminal window by click the ‚Äúx‚Äù beside ‚ÄúTerminal 1‚Äù.
